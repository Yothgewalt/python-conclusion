เริ่มต้นทำความรู้จักกับภาษาไพรธอน

จุดเริ่มต้นของการเขียนโปรแกรมที่ทุกคนต้องเจอเสมอทุกยุคทุกสมัย

ก่อนที่ทุกคนจะเริ่มเขียนโปรแกรม คุณจำเป็นต้องเริ่มติดตั้งโปรแกรมสำหรับการเขียนโปรแกรม โดยในบทเรียนนี้จะใช้เป็น Visual Studio Code ซึ่งทำให้เราใช้ชีวิตได้ง่ายขึึ้นเป็นอย่างมาก เพราะมี Autocomplete ที่ฉาญฉลาด และเราต้องศึกษาเกี่ยวกับประวัติความเป็นมาของภาษาและจุดประสงค์ของภาษานั้นๆ หลังจากนั้นเราจะทำการติดตั้ง Runtime ของตัวภาษาไพรธอน

เริ่มการติดตั้งโปรแกรม Visual Studio Code

เครื่องมือในการทำงานของวิชาคอมพิวเตอร์โปรแกรมมิ่ง

ซึ่งในขั้นตอนของการติดตั้งของโปรแกรมเป็นสกิลหลักๆ ที่ทุกคนที่เข้ามาเรียนในสาขานี้ ควรพึงจะมี ซึ่งผมจะไม่อธิบายเกี่ยวกับการติดตั้งโปรแกรมแต่ผมจะใส่ลิงก์เอาไว้ให้สำหรับกดเพื่อเข้าไปดาวน์โหลดโปรแกรม ดาวน์โหลด

เริ่มการติดตั้ง Runtime ของ Python

เครื่องมือในการทำงานของวิชาคอมพิวเตอร์โปรแกรมมิ่ง

การติดตั้ง Runtime ของ Python นั้นทำให้เราสามารถรันคำสั่งต่างๆ ของภาษาไพรธอนได้ หากเราไม่มีเราก็จะไม่สามารถรันคำสั่งของ Python ได้ ซึ่งมันจะแจ้งเตือนว่าไม่พบคำสั่งไพรธอน ซึ่งผมจะทำการวางลิงก์เอาไว้เพื่อนำคุณเข้าไปสู่เว็บไซต์ของไพรธอน​ หลังจากนั้นคุณก็จะสามารถ ดาวน์โหลด

ในระหว่างการติดตั้งแนะนำว่าให้กดติ๊กว่าให้ตัวติดตั้งไพรธอนเชื่อม PATH ให้เราด้วยนะครับ

ซึ่งในตัวของไพรธอนนั้นมีสิ่งหนึ่งที่เรียกว่า Interpreter ซึ่งเป็นตัวแปรภาษาให้กับเครื่องซึ่งคล้ายคลึงกับ Compiler ในภาษาปกติทั่วไป แต่การทำงานต่างออกไปจาก Compiler ซึ่ง Interpreter จะทำการอ่านทีละบรรทัด หากเจอบรรทัดไหนที่ไม่ถูกต้องมันก็จะแจ้งเตือนทันที แต่หากเป็น Compiler เมื่อใช้งานแล้วเราจะพบว่ามันจะแจ้งบรรทัดที่มีบรรทัดทุกอันซึ่งท่ามี 100 อัน คุณคงจะตาลาย แต่ Case ที่ผมกล่าวมานั้น เป็นแค่ส่วนหนึ่งของ Case ที่มักพบกันโดยทั่วไปและมันยังมีเยอะกว่านี้

การทักทายครั้งแรกของเรา

เริ่มต้นการเขียนโปรแกรมของคุณ

หลังจากการติดตั้งโปรแกรมที่เป็นสิ่งสำคัญเสร็จ คุณก็ต้องทำการลองเขียนโปรแกรมแรกของคุณซึ่งผมจะเสนอเป็นคำสั่งแสดงข้อความออกมาทางหน้าจอ ซึ่งผมจะใช้ฟังก์ชั่น print() พร้อมกับใส่ข้อความด้วย Double Quotes หรือ "" ตามด้วยข้อความที่อยู่ข้างในสำหรับข้อความที่จะแสดงออกมาทางหน้าจอ

ซึ่งวิธีการรันนั้น หากคุณใช้ Visual Studio Code คุณสามารถที่จะกดคลิกขวาและเลือกที่ Run Python File in Terminalซึ่งมันจะช่วยคุณในการรันไฟล์ไพรธอนแบบง่ายๆ ซึ่งคุณไม่จำเป็นต้องพิมพ์คำสั่งเพื่อรัน
